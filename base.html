HTML base

Atributos

Atributos contem informações adicionais sobre elementos. Todos os atributos são compostos de nome e valor.

  • Todos elementos html podem ter atributos.

  • Atributos são sempre especificados na tag inicial.

  • Atributos geralmente aparecem em pares de nome/valor

Cabeçalhos

  • Cabeçalhos são definidos da tag <h1> a tag h6

  • h1 define o cabeçalho mais importante. <h6> define o cabeçalho menos importante.

  • A tag só pode e deve ser usada uma única vez, as demais não há limites de uso.

Paragrafos

  • O parágrafo é o elemento base no texto editado. A tag para um parágrafo é <p>

  • O HTML adiciona automaticamente uma linha em branco extra antes e após um parágrafo.

  • Nem sempre um parágrafo é um conjunto de linhas. Existem parágrafos com somente uma palavra ou uma linha.

Estilos em html

Para estilizar elementos html é usado o atributo style, sua sintaxe é:

<nome_da_tag style="propiedade: valor;">

Sendo que "propriedade" é uma propriedade do css e "valor" um valor do css.

Pode ser usado para diversos fins:

  • Definir cor de fundo: <body style="background-color: powderblue;">

  • Alterar a cor dos textos: <h1 style="color: red;">

  • Definir a fonte dos textos: <h1 style="font-family: verdana;">

  • Alterar o tamanho dos textos: <h1 style="font-size: 300%;">

  • Definir o alinhamento do texto: <h1 style="text-align: center">

Formatação

O HTML também define elementos especiais para definir o texto com um significado especial.

O HTML usa elementos como <b> e <i> para formatação de saída, como texto em negrito ou itálico.

Elementos de formatação foram projetados para exibir tipos especiais de texto:

  • <b> - Texto em negrito
  • <strong> - texto importante
  • <i> - texto em itálico
  • <em> - texto enfatizado
  • <mark> - texto marcado
  • <small> - pequeno texto
  • <del> - texto excluído
  • <ins> - texto inserido
  • <sub> - texto subscrito
  • <sup> - Texto sobrescrito
